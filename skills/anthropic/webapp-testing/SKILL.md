# Webapp Testing

> Official Anthropic skill for validating web application behavior.
> Source: `anthropics/skills/webapp-testing`

## When to Use

Use this skill when:
- Testing endpoints exposed by MCP bridge plugins
- Validating LM Studio API responses against expected schemas
- Running integration tests for the `packages/legal` or `packages/research` tools

## Install

```bash
claude mcp add-skill anthropics/skills/webapp-testing
```
