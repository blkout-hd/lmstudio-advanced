# Web Artifacts Builder

> Official Anthropic skill for generating interactive web components and artifacts.
> Source: `anthropics/skills/web-artifacts-builder`

## When to Use

Use this skill when:
- Generating React/HTML/CSS components as Claude artifacts
- Building interactive data visualizations
- Creating embeddable UI components for legal or research dashboards
- Producing self-contained HTML reports from analysis output

## Install

```bash
claude mcp add-skill anthropics/skills/web-artifacts-builder
```
